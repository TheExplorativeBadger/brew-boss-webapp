Brewing Steps

The steps in the Step Table shown are loaded from the Brew-Boss Controller memory. You can see the default step file in the Settings tab.

Current step file:

- Use the + Icon to add a new row to the bottom of the table
- Use the x Icon in a row to delete that row from the table
- Use the Up Arrow in a row to shift the row up one spot in the table
- Use the Down Arrow in a row to shift the row down one spot in the table

Select a Beer.xml file from your file system to import as a steps file for this app.


To save any changes you have made to the currently displayed step file, please use the 'Save Steps' button directly below. *Notice: Saving any changes will overwrite the previous version of the step file.

To save an edited step file as a new step file, please enter a name for the file on the line directly below, then use the 'Save Steps As...' button. This will create a new file that can be loaded in the future. This action will leave the previous file unchanged.

The Brew-Boss Controller contains all of your previously saved step files. If you would like to load a step file, please use the dropdown list directly below to select a file from the list, then click the 'Load Steps' button.


To delete a file from those you have previously saved, please select a file from the list, and click the 'Delete File' button below. Deleting a file cannot be undone.

Settings

  • Heater / Power

Settings are saved to / loaded from the Brew-Boss Controller.

The Reset Default Settings button can be used to reset ALL settings back to their original values.


General

  • Default Step File:

    The step file that will be loaded on app startup

  • Temperature Units: F C

    Choose the temperature units you would like to use for this app

  • Heater Watts:

    Used to generate Brewing Session data

  • Cost / KW Hour ($) :

    Used to generate Brewing Session data

  • Enable AccuFill:

    Enable this to automate filling when importing Beer.xml

  • Initial AccuFill Fill Volume:

    The volume of water (in liters) to initially fill AccuFill with

  • Enable Hops Boss Feeder:

    Enable this to automate the hops additions when importing Beer.xml

  • Mortar Shell Cup Capacity (oz):

    The volume (in ounces) of the Hops-Boss Mortar Shell Cup

Power

  • Maximum Heater Power:

    When the heater is set to Max, this value will overwrite 100%

  • Boil Detect Time:

    Time in seconds required to hold boil temp. Must be greater than 45 seconds

  • Delta_T1:

    Number of degrees below Setpoint where Heater goes to PwrS

  • Delta_T2:

    Number of degrees below Setpoint where Heater goes to PwrL

  • PwrS:

    Heater Power % to use within DeltaT1 degrees of Setpoint

  • PwrL:

    Heater Power % to use within DeltaT2 degrees of Setpoint

  • PwrM:

    Heater Power % to use while maintaining temperature

  • PwrB:

    Heater Power % to use while boiling
